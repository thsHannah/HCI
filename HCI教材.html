01 · 觀念

什麼是「人機互動」？

人機互動（Human-Computer Interaction, HCI）研究的是「人」怎麼跟「電腦／機器」溝通。以前我們用鍵盤、滑鼠跟電腦說話；現在越來越多裝置可以直接理解我們的聲音、觸控、甚至肢體動作。FLOWDRAW 就是用「手勢」跟電腦互動的例子。

🎤

語音助理

你說「明天天氣如何」，手機聽懂語音、轉成文字、再回答你。

📱

觸控螢幕

手指滑動、點擊，螢幕感應到你的位置與動作，畫面隨之改變。

🎮

體感遊戲

攝影機或感應器捕捉你的動作，變成遊戲裡角色的動作。

🖐️

FLOWDRAW

攝影機拍到你的手，AI 判斷手勢，決定要畫圖、擦除還是存檔。

02 · 系統流程

FLOWDRAW 是怎麼運作的？

程式並不是「魔法」，而是把一連串明確的步驟不斷重複執行（每秒約 30 次！）。我們可以把它拆成 5 個步驟：

1

📷 攝影機拍畫面

持續讀取一張一張的畫面（影格）。

→
2

🤖 AI 找出手部

AI 模型在畫面中標出 21 個手部關鍵點座標。

→
3

🧮 計算手指狀態

比對座標與角度，判斷每根手指是「伸直」還是「彎曲」。

→
4

🔀 判斷手勢種類

用 if / else 條件式，決定現在是哪一種手勢。

→
5

🎨 執行對應動作

畫圖、擦除，並把結果顯示在畫面上。

這 5 個步驟每秒重複約 30 次，所以你揮動手指時，畫面才會「即時」跟著你動起來。

03 · 電腦視覺

電腦怎麼「看見」你的手？

FLOWDRAW 使用 Google 開發的 AI 模型 HandLandmarker。它會在畫面中的手上，標出 21 個關鍵點（手腕、每根手指的關節與指尖），每個點都有自己的 x、y 座標。這跟程式畫面右下角你會看到的紅點綠線，其實是一樣的東西！

紅點 = 關鍵點（landmark）
綠線 = 骨架連接

程式會特別關注「指尖」（例如編號 8 = 食指指尖、12 = 中指指尖）與「關節」的相對位置，來判斷手指有沒有伸直。

💡 小知識：模型用「0」代表手腕，接著每根手指各有 4 個點，5 根手指 × 4 + 1 個手腕 = 21 個點。

04 · 判斷邏輯

電腦怎麼知道手指「伸直」還是「彎曲」？

方法很簡單：比較指尖跟關節的位置。如果指尖的 y 座標比關節「高」（在畫面上數值更小），代表手指是伸直的；如果指尖幾乎跟關節疊在一起或更低，代表手指彎曲了。握拳的判斷則更精確，是去計算每根手指彎曲的角度，角度越大代表彎得越多。

伸直 → 指尖在上方

指尖 y 座標 < 關節 y 座標

彎曲 → 指尖靠近關節

指尖 y 座標 ≥ 關節 y 座標，或彎曲角度變大

05 · 手勢對照表

不同的手指組合，代表不同的指令

FLOWDRAW 把「哪些手指伸直」轉換成 3 種狀態，每種狀態對應一個動作：

手勢 1

☝️ 只伸出食指

→ 畫圖：跟著食指指尖的位置畫線。

手勢 2

✌️ 食指 + 中指

→ 擦除：以兩指中點為中心擦掉筆劃。

手勢 0

🖐️ 其他組合

→ 待機：不符合以上規則，畫面不動作。

07 · 程式設計思維

電腦真正的判斷方式：條件式

上面模擬器背後的邏輯，其實就是程式裡一段「如果…而且…就…」的條件判斷（if / elif / else）。這是所有程式設計最核心的技巧之一 —— 讓程式依照不同情況做出不同反應。以下是簡化過的版本：

如果 食指伸直 而且 中指沒伸直 而且 無名指沒伸直 而且 小指沒伸直:
    # 只有食指伸直
    手勢 = 1  → 畫圖

否則如果 食指伸直 而且 中指伸直 而且 無名指沒伸直 而且 小指沒伸直:
    # 食指 + 中指伸直
    手勢 = 2  → 擦除

否則:
    手勢 = 0  → 待機

觀察重點：程式會由上往下依序檢查每一個條件，只要符合就停止往下檢查——這就是為什麼手勢的判斷順序很重要！

08 · 實作指南

在自己電腦上動手跑起來！

學完了原理，現在要在你自己的電腦上建立 Python 虛擬環境，並啟動這個程式。請跟著以下步驟進行：

1

準備專案資料夾結構

請確保你的檔案結構如下，Flask 預設會去 templates 尋找網頁檔案：

my_flowdraw_project/
├── app.py # 後端 Python 主程式
├── static/ # 靜態檔案資料夾（可為空）
└── templates/
    └── index.html # 小程式
2

建立並啟用虛擬環境 (venv)

開啟你的終端機 (Terminal / CMD)，使用 cd 指令切換至專案資料夾，並執行以下指令：

# 1. 建立名為 venv 的虛擬環境
python -m venv venv

# 2. 啟用虛擬環境 (依作業系統選擇對應指令)
Windows (CMD): venv\Scripts\activate
Windows (PowerShell): .\venv\Scripts\Activate.ps1
macOS / Linux: source venv/bin/activate

💡 提示：啟用成功後，終端機提示字元的最前方會出現 (venv) 的標記！

3

安裝依賴套件

在啟用虛擬環境的狀態下，直接複製並執行以下指令，安裝 Flask、OpenCV 與 MediaPipe 等套件：

pip install --upgrade pip
pip install -r requirements.txt

提醒：第一次執行時程式會自動從網路下載約 8MB 的手部偵測模型。需花一點時間

4

啟動 FLOWDRAW 程式

萬事俱備！輸入啟動指令，並打開瀏覽器享受隔空畫畫的樂趣吧：

python app.py

按住ctrl/cmd點擊：http://127.0.0.1:5000
